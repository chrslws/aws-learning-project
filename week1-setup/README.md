# aws-learning-project
Programming (Black) Epiphany Term coursework

Learning AWS serverless development for Programming (Black) module at Durham University.
Focus areas will progress by week:
 - Week 1: AWS CLI introduction
 - Week 2: Using S3 on a static site
 - Week 3: Building webpages with the Lambda function
 - Week 4: Constructing an API Gateway
 - Week 5: Using DynamoDB database for storage
 - Week 6: Serverless mini project using new skills
 - Week 7: Contribution to an open source project
